2 Exemplos Simples de Histórias de Usuário com Front-end

Cada exemplo apresenta uma história de usuário, critérios de aceitação e uma implementação simples em HTML, CSS e JavaScript.

Exemplo 1: Saudação Personalizada

Como visitante, eu quero digitar meu nome para receber uma saudação personalizada.
Critérios de aceitação:
  • Campo para digitar o nome.
  • Botão para exibir a saudação.
  • Mensagem: "Olá, [nome]!".
A saudação aparecerá aqui.

Exemplo 2: Lista de Tarefas

Como usuário, eu quero adicionar tarefas para organizar minhas atividades.
Critérios de aceitação:
  • Campo para digitar a tarefa.
  • Botão para adicionar.
  • Lista atualizada em tempo real.
Nenhuma tarefa adicionada.
