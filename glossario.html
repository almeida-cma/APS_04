➕ CREATE & READ

O usuário deve digitar um Termo, sua Descrição e um Link opcional. O termo deve aparecer na lista visual em formato de "Cartão".

O usuário deve ser capaz de remover termos específicos do glossário caso estejam incorretos, usando um botão de exclusão ❌.

💾 EXPORT & IMPORT

Como o glossário fica em memória RAM temporária, o app deve permitir exportar a base de dados como `.json` para o PC e importá-la futuramente.

💻 ETAPA 7 — CÓDIGO PYTHON COMPLETO Construção de UI Dinâmica e JSON - Copie e Cole
📄 form1.py 
from anvil import *
import anvil.media
import json
import time  # Para gerar IDs únicos

class Form1(Form1Template):
  
  def __init__(self, **properties):
    self.init_components(**properties)
    
    # Memória Temporária (A Lista do Dicionário)
    self.glossario = []
    
    # Renderiza a tela vazia ao abrir o app
    self.renderizar_lista()

  # ==========================================
  # C R U D - CREATE (Adicionar Termo)
  # ==========================================
  def btn_adicionar_click(self, **event_args):
    termo_digitado = self.txt_termo.text
    desc_digitada = self.txt_descricao.text
    link_digitado = self.txt_link.text
    
    # Validação: Obriga a preencher Termo e Descrição
    if not termo_digitado or not desc_digitada:
      alert("Preencha o Termo e a Descrição!")
      return
      
    # Criação da estrutura de dados do novo item
    novo_termo = {
      "id": int(time.time() * 1000), # ID único baseado no relógio do PC
      "termo": termo_digitado,
      "descricao": desc_digitada,
      "link": link_digitado
    }
    
    # Salva na memória do Python
    self.glossario.append(novo_termo)
    
    # Limpa os campos visuais
    self.txt_termo.text = ""
    self.txt_descricao.text = ""
    self.txt_link.text = ""
    
    # Chama a reconstrução da tela
    self.renderizar_lista()

  # ==========================================
  # C R U D - READ (Montar os Cartões Visuais)
  # ==========================================
  def renderizar_lista(self):
    # Limpa tudo antes de desenhar novamente
    self.painel_termos.clear()
    
    for item in self.glossario:
      # 1. Cria a caixa principal do Cartão (LinearPanel)
      cartao = LinearPanel(role="card") 
      
      # 2. Cria o cabeçalho do cartão (Título ao lado do botão Excluir)
      linha_topo = FlowPanel(align="justify")
      
      lbl_termo = Label(text=item["termo"], bold=True, font_size=18, foreground="#0f766e")
      
      btn_del = Button(text="❌ Remover", foreground="#e11d48")
      btn_del.tag = item["id"]  # Salva o ID da palavra dentro do botão
      btn_del.set_event_handler('click', self.deletar_termo)
      
      linha_topo.add_component(lbl_termo)
      linha_topo.add_component(btn_del)
      
      # 3. Cria o texto da descrição
      lbl_desc = Label(text=item["descricao"])
      
      # Adiciona o topo e a descrição dentro do cartão principal
      cartao.add_component(linha_topo)
      cartao.add_component(lbl_desc)
      
      # 4. Se existir um link, cria o componente clicável
      if item["link"]:
        lnk = Link(text="🔗 Ver Exemplo", url=item["link"])
        cartao.add_component(lnk)
        
      # Por fim, adiciona o cartão completo no painel da tela
      self.painel_termos.add_component(cartao)


  # ==========================================
  # C R U D - DELETE (Remover do Glossário)
  # ==========================================
  def deletar_termo(self, sender, **event_args):
    # Pega o ID que guardamos no '.tag' do botão
    id_alvo = sender.tag
    
    # Sobrescreve a lista ignorando o item deletado (List Comprehension)
    self.glossario = [item for item in self.glossario if item["id"] != id_alvo]
    
    # Redesenha a tela para o cartão sumir
    self.renderizar_lista()


  # ==========================================
  # JSON - EXPORTAR (Baixar Dicionário)
  # ==========================================
  def btn_exportar_click(self, **event_args):
    # Converte a Lista de Dicionários para formato de texto JSON
    json_texto = json.dumps(self.glossario, indent=4)
    
    # Empacota como um arquivo para o navegador baixar
    arquivo = anvil.BlobMedia('application/json', json_texto.encode(), name='meu_glossario.json')
    anvil.media.download(arquivo)


  # ==========================================
  # JSON - IMPORTAR (Upload do FileLoader)
  # ==========================================
  def fl_importar_change(self, file, **event_args):
    # Lê o arquivo que o usuário subiu e converte para texto
    conteudo_texto = file.get_bytes().decode('utf-8')
    
    try:
      # Carrega o JSON substituindo o glossário atual e atualiza a tela
      self.glossario = json.loads(conteudo_texto)
      self.renderizar_lista()
    except Exception as e:
      alert("Erro: O arquivo não é um JSON de glossário válido!")
📖 ETAPA 8 — EXPLICAÇÃO DETALHADA DO CÓDIGO Os segredos do código dinâmico

🧱 Blocos Aninhados (Nesting)

Diferente da To-Do List, aqui nós aninhamos painéis. Criamos um LinearPanel(o Cartão). Dentro dele, colocamos o Cabeçalho (FlowPanel), depois a Descrição, e depois o Link. A UI é criada como blocos de Lego empilhados.

🔗 A Mágica do .tag e do ID

Como saber qual botão "Remover" deleta qual termo? Geramos um ID usando o relógio time.time(). Guardamos esse ID na propriedade .tag do Botão. No clique, lemos o tag e apagamos o termo exato da memória!

🧹 O padrão Delete em Python

No deletar_termo, nós usamos *List Comprehension*. A regra é: "A nova lista é igual à antiga, MAS SÓ COPIE os itens cujo ID seja DIFERENTE do botão clicado." Isso extingue a palavra removida.

📤 json.dumps() e BlobMedia

json.dumps() "Despeja" (dumps) código Python virando texto puro em formato JSON. O BlobMedia empacota esse texto e avisa ao navegador: "Isso é um arquivo .json, inicie o download!".

📥 json.loads() e FileLoader

O FileLoader captura arquivos do PC. Com file.get_bytes() lemos o arquivo criptografado, decodificamos para texto, e o json.loads() "Carrega" (loads) o texto transformando-o novamente em uma Lista Python!
